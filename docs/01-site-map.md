# EzHome — Site Map (Shopify)

> Complete page hierarchy for the EzHome Shopify e-commerce store.
> Products: Compression sofas & beds — lightweight, 10-minute assembly.

---

## Shopify URL Structure

Shopify enforces specific URL patterns. All routes below follow Shopify conventions.

---

## 1. Storefront Pages (Public)

```
/                                          ← Homepage (index.liquid)
│
├── /collections                           ← All Collections page
│   ├── /collections/sofas                 ← Sofas collection
│   ├── /collections/beds                  ← Beds collection
│   ├── /collections/accessories           ← Covers, Legs, Accessories
│   ├── /collections/new-arrivals          ← Auto: tag "new-arrival"
│   ├── /collections/best-sellers          ← Auto: best-selling sort
│   ├── /collections/limited-edition       ← Manual: curated drops
│   └── /collections/all                   ← Shopify auto: all products
│
├── /products/:handle                      ← Product Detail Page (PDP)
│   (e.g. /products/compression-sofa-2-seater)
│   In-page sections:
│   ├── #overview                          ← Hero gallery + configurator
│   ├── #how-it-works                      ← Compression & assembly
│   ├── #specs                             ← Specifications table
│   ├── #reviews                           ← Customer reviews (Judge.me / Shopify)
│   └── #faq                               ← Product-specific FAQ
│
├── /pages/how-it-works                    ← How Compression Technology Works
├── /pages/about                           ← Our Story
├── /pages/sustainability                  ← Materials & Sustainability
├── /pages/reviews                         ← All Customer Reviews
├── /pages/contact                         ← Contact Us (Shopify form)
├── /pages/faq                             ← Frequently Asked Questions
├── /pages/shipping-returns                ← Shipping & Returns Policy
├── /pages/warranty                        ← Warranty Information
├── /pages/care-guide                      ← Product Care Guide
├── /pages/assembly                        ← Assembly Instructions (video + steps)
├── /pages/press                           ← Press / Media Kit
├── /pages/careers                         ← Careers
│
├── /blogs/journal                         ← Blog (Shopify blog object)
│   ├── /blogs/journal/tagged/:tag         ← Blog filtered by tag
│   └── /blogs/journal/:handle             ← Blog article detail
│
├── /policies/terms-of-service             ← Terms (Shopify legal pages)
├── /policies/privacy-policy               ← Privacy Policy
├── /policies/refund-policy                ← Refund Policy
├── /policies/shipping-policy              ← Shipping Policy
│
├── /gift_cards/:id                        ← Gift Card page (Shopify native)
├── /search                                ← Search results page
├── /sitemap.xml                           ← Auto-generated by Shopify
└── /404                                   ← Not Found
```

## 2. Shopping & Checkout

```
/cart                                      ← Cart page (also: AJAX cart drawer)
│
/checkouts/:token                          ← Shopify Checkout (hosted)
│   Step 1: Contact + Shipping address
│   Step 2: Shipping method
│   Step 3: Payment (Shopify Payments / Stripe / Afterpay)
│
/checkouts/:token/thank_you               ← Order confirmation
│
/tools/recurring/confirm/:token            ← Subscription confirm (if using Recharge)
```

> **Note:** Shopify checkout is hosted and cannot be fully customized on standard plans. Shopify Plus enables checkout.liquid / Checkout Extensibility.

## 3. Customer Account Pages

```
/account                                   ← Account Overview (Shopify customer)
├── /account/login                         ← Login
├── /account/register                      ← Registration
├── /account/reset/:token                  ← Password Reset
├── /account/activate/:token               ← Account Activation
│
├── /account                               ← Order history & profile
│   Sections (within account.liquid):
│   ├── Order History                      ← List of past orders
│   ├── Account Details                    ← Name, email, phone
│   └── Saved Addresses                    ← Address book
│
└── /account/orders/:id                    ← Individual order detail
```

> **Note:** Shopify's native customer accounts are limited. For wishlists, referrals, and advanced features, use apps (e.g., Wishlist King, Smile.io).

## 4. Admin (Shopify Admin Dashboard)

```
/admin                                     ← Shopify Admin (admin.shopify.com)
├── /admin/orders                          ← Order management
├── /admin/products                        ← Product management
├── /admin/customers                       ← Customer management
├── /admin/discounts                       ← Discount codes & automatic discounts
├── /admin/analytics                       ← Sales & traffic analytics
├── /admin/marketing                       ← Campaigns, automations
├── /admin/content                         ← Blog posts, pages (metaobjects)
├── /admin/themes                          ← Theme editor (Online Store 2.0)
├── /admin/settings                        ← Store settings, payments, shipping
└── /admin/apps                            ← Installed apps
```

> Admin is fully managed by Shopify. No custom admin panel needed.
