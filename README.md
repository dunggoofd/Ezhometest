# Compression Sofa - Premium Ecommerce Website

A modern, high-performance ecommerce website built for premium compression sofas that ship compact and assemble in 10 minutes.

## Features

### 🎥 Video-Driven Product Showcase
- Hero section with dynamic video backgrounds
- Art-directed responsive images (mobile & desktop)
- Lazy loading and optimized media delivery

### 🎨 Modular Product Configuration
- Interactive size selection (Loveseat, Standard, Large)
- Premium fabric options (Linen, Velvet, Italian Leather)
- Multiple color choices with live preview
- Add-on accessories and bundles

### 📦 Product Bundling System
- Curated bundles with significant savings
- Custom bundle builder
- Real-time price calculations

### 🏠 Virtual Showroom
- 3D product visualization using Three.js
- Interactive room placement
- AR view support for mobile devices
- 360° product rotation

### 💰 Conversion-Optimized Sales Engine
- Social proof and testimonials
- Urgency & scarcity indicators
- Exit-intent popup with discount offer
- Trust signals and guarantees
- Multiple CTAs throughout the journey

### 📱 Mobile-First Responsive Design
- Art-directed images for different breakpoints
- Touch-optimized interactions
- Fast loading with Core Web Vitals optimization

### 🔍 SEO Optimizations
- Schema.org structured data (Product, WebSite, Organization)
- hreflang tags for international markets
- Open Graph and Twitter Card metadata
- Semantic HTML and accessibility features
- Optimized for Core Web Vitals

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **3D Graphics**: Three.js, React Three Fiber, Drei
- **Animation**: Framer Motion
- **Video**: React Player
- **Ecommerce**: Shopify Hydrogen React (integration ready)

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
